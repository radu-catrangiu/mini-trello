## Dependencies

The project depend on:
* npm
* node
* vue-cli 


## Project setup
```
In project root:

    npm install
    cd backend
    npm install
```

## Project start
The command below is for UNIX-like systems. On Windows you must change directory to './backend' and run 'node index'
```
In project root:

    npm run start
```
